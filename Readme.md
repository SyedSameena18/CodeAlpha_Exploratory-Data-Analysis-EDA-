# Exploratory Data Analysis of Global Layoffs Dataset

## Project Overview

This project presents an Exploratory Data Analysis (EDA) of a global layoffs dataset containing workforce reduction records from companies across multiple industries and countries. The objective is to identify trends, patterns, and insights related to layoffs, company performance, industry impact, and geographical distribution.

## Objectives

* Understand the structure and quality of the dataset.
* Perform data cleaning and preprocessing.
* Analyze layoff trends across companies, industries, and countries.
* Identify significant patterns and anomalies in workforce reductions.
* Generate actionable insights through statistical analysis.

## Dataset Information

The dataset contains information about:

* Company Name
* Location
* Industry
* Total Employees Laid Off
* Percentage of Workforce Laid Off
* Funding Raised
* Company Stage
* Country
* Date of Layoff

## Key Findings

* The dataset contains 4,453 layoff records from 2,925 companies.
* The United States recorded the highest number of layoff events.
* Certain industries experienced significantly larger workforce reductions than others.
* Layoff activity increased substantially during specific periods.
* Multiple companies reported laying off their entire workforce.

## Conclusion

The analysis highlights the widespread impact of layoffs across industries and regions. By examining workforce reductions from multiple perspectives, the project provides valuable insights into economic trends, organizational restructuring, and industry-specific challenges.
